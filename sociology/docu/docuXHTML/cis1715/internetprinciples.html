Internet Principles

 

1)      What protocol is used by all computer connections to the Internet?

TCP/IP is the communication protocol used by all computer connections to the Internet. File Transfer Protocol is used by all computer connections to send and receive files. Hypertext Transfer Protocol is not used by all computers because web browsers and web servers are required. Similarly, E-mail Protocol (SMTP/IMAP) requires mail servers.

 

2)      What is the form of an IP Address?

The IP Address is a numeric sequence denoting the location of a website. It is the ‘number’ version of a web address. The IP is numeric-based and corresponds to the domain name’s URL, which is text-based.

 

3)      Describe a fully qualified domain name.

A domain name merely locates entities on the Internet through the Domain Name System. However, a fully-qualified domain name includes the web host server (www) plus the second-level domain name registered to the host. In other words, it is the complete domain name: www.truthdig.com.

 

4)      What is the task of the DNS name server?

The Domain Name System name server is a software system that converts the text-based URL into the numeric-based IP address, and vice versa.  We type out, “facebook.com” and the name server translates it into the IP address that resembles xxx.xxx.xxx.xxx.

 

5)      What is the purpose of a Domain Name?

The purpose of a Domain Name is to divide the Internet into ‘logical groups and understandable names by identifying the exact address and type of the organization.’ So, www.cnm.edu identifies the location (CNM) and the type (accredited educational organization).

 

6)      List and describe the different parts of a URL.

URL stands for Uniform Resource Locator. The URL http://www.unm.edu/alumni/2012reunion.html includes the hypertext transfer protocol (http), the web server computer name (www), the domain name (unm.edu), the folder name (/alumni), and web page name(/2012reunion.html). 

 

7)      Describe the purpose of the Transmission Control Protocol

TCP is used to divide up files and messages into individual packets, so that each packet contains information such as destination, source, and checksums. This is to maintain the integrity of the data transmission.

 

8)      Describe the purpose of the Internet Protocol.

IP works in tandem with TCP as the ‘set of rules’ that dictates how these TCP ‘data packets’ are sent.  This ensures that the packet gets forwarded, router by router, to its destination IP Address.
